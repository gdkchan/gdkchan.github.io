VideoLauncher


Sumário

O VideoLauncher é um aplicativo escrito em VB.NET/WPF para procurar, assistir e baixar vídeos de sites populares, tais como o Youtube, UOL Mais, Vimeo e Daily Motion. Outros recursos como suporte a listas de reprodução e comentários também são ou serão suportados no futuro.


Funcionamento

Para a extração de vídeos, a página que contém o vídeo é baixada, e então, atrávés de expressões regulares (RegEx), os dados relevantes para o aplicativo são extraídos do código HTML/JavaScript da página. No caso da extração, os dados relevantes são os links para os Video Streams, que são basicamente links diretos para os vídeos, usados pelo player para fazer a reprodução do mesmo. Estes links também são usados para se fazer o download dos vídeos. No caso do Youtube, as informações relevantes também incluem uma assinatura usada para proteger vídeos do VEVO (entre outros), e as ITags, que são códigos próprios do Youtube que determinam o formato, resolução e qualidade em geral do vídeo. O VideoLauncher é capaz de "desembaralhar" as assinaturas dos vídeos do VEVO, tornando possível a sua reprodução fora do Youtube, e também o download destes vídeos.


As buscas também extraem informações relevantes da página HTML através de RegEx. As informações extraídas na busca são o Título, descrição, duração, miniatura (no caso do Youtube, também uma imagem de alta resolução) e contagem de visualizações do vídeo. Note que nem todos os sites possuem todas essas informações para os vídeos. Nesses casos, o VideoLauncher deixará os campos vazios.
